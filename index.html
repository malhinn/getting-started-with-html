Mohammad's Notes Intro to the Web

The web is made up of HTML pages.

HTML

HTML is the basic language that undergirds all web content. It is made up of text content, markup, and references to other documents.

Text Content

What we see on a web page.

Markup

Tags, e.g., bold.

References to Other Documents

In the form of tags with HTML attributes, e.g., links to web pages, images.

Specific Tags

Bold: [b][/b]

Italics: [em][/em]

Underline: [u][/u]

¶: [p][/p]

↵: [br][/br]

Span: [span][/span]

Div: [div][/div]

Anchor: [a][/a]

: [img src="url"]
